# Boris WEMBE — GitHub Pages Website

This repository contains a static academic website designed for GitHub Pages.

## Files

- `index.html` — main page
- `styles.css` — visual theme and layout
- `script.js` — mobile navigation and small interactions
- `assets/Boris_WEMBE_CV.pdf` — replace with the final CV PDF

## Publish on GitHub Pages

1. Create or use the repository `wboris03.github.io`.
2. Upload all files from this folder to the repository root.
3. In GitHub, open **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and the `/ (root)` folder.
6. Save.

Your site will then be available at:

`https://wboris03.github.io`
